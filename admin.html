    Apparence & sécurité

    Publier votre visite

    Vos modifications sont enregistrées dans ce navigateur. Pour les mettre en ligne sur votre site, téléchargez le pack puis envoyez-le sur GitHub.

    1. Décompressez le fichier .zip téléchargé.
    2. Copiez son contenu (tour.json + dossier photos/) dans votre dossier photo360, en remplaçant les fichiers existants.
    3. Dans Git Bash, depuis le dossier photo360, lancez : 
      git add -A
git commit -m "Mise a jour de la visite"
git push origin main
    4. Patientez ~1 min : votre site est à jour sur https://seb0031.github.io/photo360/
